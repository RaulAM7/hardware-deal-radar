# Hardware Deal Radar

Internal radar for finding real buying opportunities in used or refurbished professional ThinkPad laptops on eBay Europe.

This repo currently contains the project context and planning scaffold. It is prepared for a planning/spec pass before implementation.

## Product Direction

- Build a VPS-first CLI system, not a web app.
- Query the official eBay Browse API for ES, DE, IT, and GB marketplaces.
- Normalize listings, estimate total cost, deduplicate, score deterministically, persist to SQLite, and alert only when signal is strong.
- Send strong opportunities by Telegram and medium candidates by email digest.
- Support real, mock, dry-run, and noop modes so missing credentials never block development.

## Closed v1 Decisions

- Language: Python 3.12+
- Persistence: SQLite
- Config: YAML
- Interface: CLI
- Source: official eBay Browse API
- Alerts: Telegram plus email digest
- Scheduling: systemd timer every 6 hours
- Deployment: VPS with venv plus systemd
- Docker: optional/documented, not required
- Scope: ThinkPad only in v1
- Minimum RAM: 32 GB
- Ideal RAM: 64 GB
- Scoring: deterministic and explainable, no generative AI in v1
- Out of scope: frontend, SaaS, scraping, automatic purchase, Postgres/Supabase, Kubernetes

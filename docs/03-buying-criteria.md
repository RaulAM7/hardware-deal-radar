# 03 — Buying Criteria

## Perfil de compra

El objetivo es encontrar un portátil business, preferiblemente ThinkPad, que suponga una mejora real frente al equipo actual del usuario.

El uso previsto incluye:

- desarrollo de software;
- sesiones simultáneas de Codex/Claude Code;
- múltiples worktrees;
- navegador con muchas pestañas;
- herramientas de productividad;
- posible ejecución de servicios locales;
- Docker/containers ligeros;
- tareas de negocio, ventas, CRM, documentación y automatización;
- Linux como entorno principal.

## Requisitos mínimos

Un listing debe ser descartado o fuertemente penalizado si no cumple:

- RAM mínima: 32 GB.
- Equipo no gamer.
- Chasis business/profesional.
- CPU razonablemente moderna.
- Estado funcional claro.
- Precio total estimado razonable.
- Vendedor sin señales graves de riesgo.
- Envío viable a España/Canarias o alternativa clara.

## Requisitos ideales

Priorizar listings con:

- 64 GB RAM.
- SSD 1 TB.
- CPU AMD Ryzen PRO moderna o Intel i7/i9 moderno eficiente.
- Buen soporte Linux.
- Peso razonable.
- Buen teclado.
- Pantalla decente.
- Vendedor profesional.
- Alta valoración.
- Política de devolución.
- Garantía o reacondicionado profesional.
- Descripción técnica clara.
- Fotos reales.
- Precio por debajo del mercado visible.

## Familias prioritarias v1

### ThinkPad P14s

Muy interesante si aparece con 64 GB y CPU moderna. Buen equilibrio entre portabilidad, potencia y enfoque profesional.

### ThinkPad T14

Interesante si tiene RAM suficiente o ampliable. Priorizar generaciones modernas.

### ThinkPad T16

Interesante si el usuario acepta algo más de tamaño a cambio de pantalla y ergonomía. Penalizar si pesa demasiado o el precio no compensa.

### ThinkPad P1 / X1 Extreme

Solo si aparece una ganga real. Puede ofrecer más potencia, pero también más calor, peso, ruido y consumo. Debe pasar un filtro más exigente.

## Fuera de foco v1

- Gaming laptops.
- Workstations demasiado pesadas.
- Equipos con menos de 32 GB.
- Equipos muy antiguos aunque tengan mucha RAM.
- Equipos con GPU dedicada problemática si penaliza ruido/calor/autonomía.
- Equipos con precio similar a nuevo sin ventaja clara.
- Equipos sin información suficiente.

## Teclado

El teclado no español no descarta automáticamente.

Debe penalizarse según riesgo:

- ES: sin penalización.
- UK: leve o media, según preferencia.
- DE/IT: media.
- Teclado no identificado: media.
- Teclado raro/no estándar: fuerte.

El scoring debe explicar claramente este riesgo.

## Reino Unido

UK no debe descartarse, pero debe penalizarse por:

- aduanas;
- fricción de importación;
- posible coste total superior al visible;
- tiempos;
- devoluciones más complejas;
- incertidumbre fiscal.

El sistema no debe prometer cálculo fiscal exacto. Debe estimar y advertir.

## Precio

El precio evaluado debe ser coste total estimado, no solo precio del listing.

Coste total estimado:

```txt
precio base + envío + penalizaciones/estimaciones por importación/riesgo
```

El sistema debe distinguir:

- precio visible;
- envío;
- coste total estimado;
- penalización de riesgo no monetaria.

## Señales positivas

- 64 GB RAM.
- 1 TB SSD.
- CPU moderna.
- Vendedor business.
- Feedback alto.
- Devolución aceptada.
- Garantía.
- Fotos reales.
- Descripción completa.
- Precio bajo.
- País UE.
- Modelo reciente.
- RAM ampliable si no trae 64 GB.

## Señales negativas

- RAM < 32 GB.
- Precio total alto.
- Vendedor con feedback bajo.
- Sin devolución.
- Descripción pobre.
- Sin fotos reales.
- Teclado no claro.
- UK sin claridad de costes.
- Equipo antiguo.
- Batería dudosa.
- Pantalla dañada.
- "For parts", "spares", "defective", "BIOS locked", "no boot".
- Subasta con precio aún no representativo.
